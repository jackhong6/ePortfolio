The internet is just a whole bunch of computers connected in a network! The computers with webpages stored on them are called servers, and they serve other computers the web pages in HTML format. The computers requesting the webpages from the servers are called clients.

There are three main languages used in web pages:

  • HTML - Hypertext Markup Language. This is for the layout of the page.
  • CSS - Cascading Style Sheet. This is for making the webpage pretty.
  • JS - Javascript. This is for making the webpage interactive.

HTML marks up text with tags. Tags are special text enclosed with angle brackets. For example, <html>. Most tags need to be opened, and then closed, but not all. For example, the <br> tag does not need to be closed.
